# OpinionatedContactPage

[resonitemodloader](https://github.com/resonite-modding-group/ResoniteModLoader) mod that provides an opinionated sorting to the contacts page. the order is:

1) unread messages/invites
2) users you can join, regardless of status
3) users you can't join, sorted by online status
4) headless users
5) contact requests
6) offline users

ties are broken alphabetically rather than last message time
